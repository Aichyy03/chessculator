# ♟️ ChessCulator

ChessCulator, satranç maçlarındaki ELO değişimlerini FIDE standartlarına (K-Katsayısı) göre hesaplayan, geçmiş maçları kaydeden ve genel reyting gidişatını analiz eden modern bir web uygulamasıdır. 

## 🚀 Canlı Demo
Projeyi indirmeden, doğrudan tarayıcı üzerinden test etmek için: 
**👉 [ChessCulator Canlı Yayın Linki](https://chessculator.netlify.app/)**

## ✨ Özellikler
- **Dinamik ELO Hesaplama:** FIDE standartlarına uygun (K: 10, 20, 40) olarak kazanma, kaybetme ve beraberlik durumlarına göre anlık ELO değişimi hesaplaması.
- **Veri Kalıcılığı (Local Storage):** Girilen tüm maç geçmişi tarayıcının yerel hafızasında tutulur. Sayfa yenilense veya kapatılsa bile veriler kaybolmaz.
- **Modern ve Ferah Arayüz:** Kullanıcı deneyimi (UX) ön planda tutularak, Tailwind CSS ile tasarlanmış modern "SaaS" arayüzü.
- **Hızlı Veri Girişi:** Hızlı analiz için gereksiz form detaylarından arındırılmış, anında sonuç veren kayıt sistemi.

## 🛠️ Kullanılan Teknolojiler
- **Kütüphane:** React (Vite altyapısı ile)
- **Stil & Tasarım:** Tailwind CSS
- **İkonografi:** Lucide React
- **Veri Yönetimi:** Tarayıcı LocalStorage API
